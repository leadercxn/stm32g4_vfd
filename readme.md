# STM32G474 开发板调试三相变频器项目

1. 工程使用 `Makefile + GCC` 工具链进行工程编译，烧录
2. 工程目录结构
```
    |-- app
    |-- dh_sdk
```

3. 编译
```sh
    make
```

4. 烧录
```sh
    make flash
```

# 硬件
1. U7-74HC244D , 输入输出同高同低
2. U3-74LVC06A , 输入取反到输出
3. 电压公式转换（举例）：  IU * 3/4 = ADCA2   =====> ADCA2 * 4 / 3 = IU = A2adc / 4095 * 3.3 * 4 / 3
    3.1. 用正点原子的驱动板： IU = A2adc / 4095 * 3.3 * 4 / 3 = Ii * 0.12
         ===> Ii = A2adc / 4095 * 3.3 * 4 / 3 / 0.12
